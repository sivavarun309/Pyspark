Python Miscellaneous (Interview Notes)

1. __slots__

Definition

Limits object attributes and saves memory.

Syntax

Python
class Student:
    __slots__ = ("name", "age")

Example

Python
s = Student()
s.name = "Tom"
s.age = 20
# s.city = "NY" ❌ AttributeError

Use: Large number of objects.

2. NamedTuple

Definition

Tuple with named fields.

Syntax

Python
from collections import namedtuple

Person = namedtuple("Person", ["name", "age"])

Example

Python
p = Person("John", 25)
print(p.name)

Output

John

Immutable like tuple.

3. Enum

Definition

Represents fixed constant values.

Syntax

Python
from enum import Enum

class Color(Enum):
    RED = 1
    BLUE = 2

Example

Python
print(Color.RED)

4. Dataclass

Definition

Automatically creates constructor, repr, equality methods.

Syntax

Python
from dataclasses import dataclass

@dataclass
class Student:
    name: str
    age: int

Example

Python
s = Student("Tom", 20)
print(s)

5. Descriptor

Definition

Controls attribute access using methods.

Methods

Python
__get__()
__set__()
__delete__()

Example

Python
class Descriptor:
    def __get__(self, obj, objtype):
        return "Hello"

class Test:
    x = Descriptor()

print(Test().x)

6. Metaclass

Definition

Class that creates other classes.

Syntax

Python
class Meta(type):
    pass

class A(metaclass=Meta):
    pass

Used for: Frameworks (Django ORM, SQLAlchemy)

7. Monkey Patching

Definition

Modify class/function at runtime.

Example

Python
class A:
    def show(self):
        print("Old")

def new_show(self):
    print("New")

A.show = new_show

A().show()

Output

New

8. Deep Copy

Definition

Creates completely independent copy.

Syntax

Python
import copy

b = copy.deepcopy(a)

Example

Python
a = [[1,2]]
b = copy.deepcopy(a)

b[0][0] = 100

print(a)

Output

[[1,2]]

9. Shallow Copy

Definition

Copies outer object only.

Syntax

Python
import copy

b = copy.copy(a)

Example

Python
a = [[1,2]]
b = copy.copy(a)

b[0][0] = 100

print(a)

Output

[[100,2]]

10. copy module

Provides copy utilities.

Python
import copy

copy.copy(obj)       # Shallow
copy.deepcopy(obj)   # Deep

11. Singleton Pattern

Definition

Only one object is created.

Example

Python
class Singleton:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance

a = Singleton()
b = Singleton()

print(a is b)

Output

True

12. Factory Pattern

Definition

Creates objects without exposing creation logic.

Example

Python
class Car:
    pass

class Bike:
    pass

class Factory:
    def create(self, vehicle):
        if vehicle == "car":
            return Car()
        return Bike()

obj = Factory().create("car")

13. Strategy Pattern

Definition

Choose algorithm at runtime.

Example

Python
class Add:
    def execute(self,a,b):
        return a+b

class Multiply:
    def execute(self,a,b):
        return a*b

strategy = Add()
print(strategy.execute(2,3))

14. Observer Pattern

Definition

One object notifies multiple objects.

Example

Python
class Observer:
    def update(self):
        print("Updated")

class Subject:
    def __init__(self):
        self.obs=[]

    def attach(self,o):
        self.obs.append(o)

    def notify(self):
        for i in self.obs:
            i.update()

s = Subject()
s.attach(Observer())
s.notify()

16. Dependency Injection (DI)

Definition

Pass dependencies instead of creating them inside a class.

Example

Python
class Engine:
    pass

class Car:
    def __init__(self, engine):
        self.engine = engine

car = Car(Engine())

Benefit: Easier testing and loose coupling.
